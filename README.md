# W2-Homework4
🚗 GetWash Landing Page

GetWash is a modern and responsive landing page for a car wash booking application. The website is designed to help users quickly find the nearest available car wash, book a convenient time, and manage their reservations through a mobile app.

✨ Features
📱 Modern and clean UI inspired by mobile app design.
🔍 Hero section with a clear call-to-action and app preview.
⚡ "Quick Start" section explaining how the service works step by step.
🤝 Partner section highlighting benefits for businesses joining the platform.
📥 Download section with App Store and Google Play buttons.
⭐ Customer reviews/testimonials section.
📱 Fully responsive layout optimized for desktop, tablet, and mobile devices.
🛠️ Built With
HTML5
Tailwind CSS
Responsive Design (Mobile First)
Flexbox & CSS Grid
🎯 Project Goal

The main objective of this project was to practice building a modern landing page using Tailwind CSS, improve responsive web design skills, and recreate a professional UI from a design mockup.

📚 What I Practiced
Creating responsive layouts with Tailwind breakpoints (sm, md, lg).
Using Flexbox and Grid for page structure.
Styling components with utility classes.
Working with custom colors, shadows, spacing, and typography.
Building reusable UI sections and maintaining clean code organization.
